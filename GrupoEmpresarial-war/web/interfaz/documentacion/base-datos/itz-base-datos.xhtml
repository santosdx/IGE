Estas tres tablas principales almacenan la información sobre los grupos empresariales, unidad legal y establecimientos; Guardando la relación historia mediante registros únicos en cada actualización que se realiza con el formulario electrónico.

Las tablas referenciales almacenan todos los datos de referencia necesarios para la actualización de la información mediante el formulario Web; Tablas como departamento, municipio, país, tipo de unidad, situación de control entre otras.

Las tablas del aplicativo son las que contienen los parámetros para el funcionamiento del formulario web; parametrización de roles, usuarios, permisos y demás.
